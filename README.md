# 6-Month AI Engineering & MLOps Roadmap

A structured, hour-calculated curriculum — from data foundations to production-ready AI and MLOps engineering.

> 1 hour of video ≈ 2–3 hours of hands-on coding and debugging.

---

## Overview

| Month | Focus | Video Hours |
| :--- | :--- | ---: |
| Month 1 | Databases, Backend, ML Introduction | 22.5h |
| Month 2 | Machine Learning & Deep Learning Intro | 24.0h |
| Month 3 | Deep Learning Intensive | 39.0h |
| Month 4 | Computer Vision & LLM Foundations | 18.5h |
| Month 5 | RAG Systems & LLM Applications | 17.5h |
| Month 6 | AI Agents, MLOps, DevOps, Microservices | 33.0h |
| **Total** | | **~154.5h** |
